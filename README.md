# vue-chat-app
Just a simple chat app that use
- firebase backend
- rxjs
- vuex for state management

## Features
- Login, Register
- One2one chat


## How to run
### Environment
- Change `.env.example` to `.env`
- Update `.env` with your firebase config

### Development
```
yarn serve
```

### Production
```shell script
yarn build
```

### Unit test
```shell script
yarn test:unit
```
